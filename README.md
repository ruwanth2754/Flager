# *Flager*
<div align="center">
<img src="./Flager.png"/>
</div>
Flager is a web reconnaissance tool written in Go that helps you discover hidden web pages and endpoints. It sends HTTP requests to a target website and analyzes the responses based on status codes, allowing you to filter results. Flager is designed for security testing and bug bounty hunting, offering features like customizable wordlists, concurrency control, and response filtering. It's a powerful tool for identifying hidden directories, admin panels, and other sensitive areas of a website.
